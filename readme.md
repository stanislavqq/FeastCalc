# Чек‑анализатор: Excel из QR‑чека
Простое приложение для автоматического создания Excel‑таблицы на основе данных из кассового чека (по QR‑коду) с возможностью распределения расходов между участниками.

## Что делает приложение

- Сканирует QR‑код с кассового чека (или принимает его фотографию/скан).
- Отправляет запрос в ФНС для получения детальной информации о покупке.
- Создаёт Excel‑таблицу по заранее подготовленному шаблону.
- Заполняет таблицу списком товаров и их стоимостью из чека.
- Позволяет распределить товары между участниками (например, если платил один человек за компанию).

## Основные возможности

- Автоматическое извлечение данных из QR‑кода чека.
- Интеграция с API ФНС для получения информации о чеке.
- Заполнение Excel‑таблицы по пользовательскому шаблону.
- Ручное назначение каждого товара конкретному участнику.
- Экспорт итогового расчёта в Excel для удобного учёта расходов.
